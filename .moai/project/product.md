# Music Trainer 프로젝트

## 프로젝트 개요

**프로젝트명**: Music Trainer (Guitar MP3 Trainer v2)

**프로젝트 설명**: 악기 연주자와 음악 학습자를 위한 웹 기반 오디오 플레이어입니다. 섹션 반복, 속도/피치 제어, 악기별 음원 분리를 통해 효율적이고 집중된 음악 연습 환경을 제공합니다.

**대상 사용자**: 악기 연주자, 음악 학습자, 음악 교육자

**핵심 가치 제안**: 음악 연습을 위한 전문 도구로서 다음을 제공합니다:
- 섹션 반복 재생(A-B 루프)으로 어려운 부분 집중 연습
- 속도/피치를 완전히 독립적으로 제어하여 자신의 수준에 맞춘 학습
- 악기별 음원 분리로 합주 부분 연습 및 음악 이해도 증진
- 직관적인 인터페이스와 프로 음악 앱 스타일의 다크 테마

## 핵심 기능

### Phase 1: MVP - 기본 재생 및 파형 시각화

**오디오 로딩**
- 드래그 앤 드롭으로 오디오 파일 로드
- 파일 선택 대화창을 통한 로드
- YouTube URL 지원(Phase 1 범위에서 평가)

**기본 재생 제어**
- 재생/일시정지/정지 버튼
- 파형을 클릭하여 직접 위치 조정 가능한 시크 기능
- 현재 시간 및 전체 길이 표시

**파형 시각화**
- 실시간 파형 렌더링(wavesurfer.js 7.x)
- 재생 위치 표시(재생 헤드)
- 빠른 스크롤 성능

**볼륨 제어**
- 슬라이더 기반 마스터 볼륨 조절
- 볼륨 백분율 표시
- 뮤트/언뮤트 기능

**A-B 루프**
- 파형 클릭으로 A 지점 설정
- 다시 클릭으로 B 지점 설정
- 루프 영역 시각적 강조
- 루프 활성화/비활성화 토글

**키보드 단축키**
- 스페이스: 재생/일시정지
- 화살표 좌/우: 5초 뒤로 가기/앞으로 가기
- I/O: A-B 루프 지점 설정
- M: 뮤트 토글

### Phase 2: 고급 속도/피치 제어

**속도 제어(피치 독립)**
- 0.5x ~ 2.0x 범위에서 조절 가능
- 0.01x 단위의 정밀 제어 (+/- 버튼 방식)
- 실시간 스트리밍 적용 및 무끊김 재생
- soundtouchjs 라이브러리 기반 실시간 처리

**피치 제어(속도 독립)**
- -12 ~ +12 반음 범위
- 반음 단위의 정밀 제어
- 원래 재생 속도 유지
- 음악 키를 변경하여 난이도 조절 가능

**무끊김 재생**
- 속도/피치 변경 시에도 오디오 끊김 없음
- 부드러운 트랜지션
- ScriptProcessorNode + SimpleFilter 기반 실시간 스트리밍

### Phase 3: 악기별 음원 분리 및 파트별 볼륨 제어

**음원 분리(음악 소스 분리)**
- Demucs 기술을 활용한 AI 기반 악기별 분리
- 주요 음원: 보컬, 드럼, 베이스, 기타, 피아노 등
- 클라우드 기반 처리(Python FastAPI + Demucs)
- 분리된 음원 캐싱으로 재사용 시간 단축

**파트별 볼륨 제어(스템 믹서)**
- 각 악기별 독립적 볼륨 조절
- 특정 악기를 음소거하여 다른 파트 집중 학습
- 합주 연습 시 각 악기의 밸런스 조정
- 실시간 프리뷰

**스템 믹서 UI**
- 각 악기의 슬라이더(보컬, 드럼, 베이스, 기타 등)
- 개별 음소거 버튼
- 현재 볼륨 백분율 표시
- 프리셋 저장/로드 기능

## 사용 시나리오

### 기본 워크플로우: 혼자 악기 연습

사용자는 Music Trainer에 오디오 파일을 로드합니다. 파형을 확인하면서 어려운 부분을 A-B 루프로 표시하여 반복 재생합니다. 음악이 너무 빠르면 속도를 0.8x로 늦춥니다. 자신의 악기 음역대에 맞추기 위해 피치를 +2 반음으로 올립니다. 루프 부분을 5번 반복하여 연습한 후, 정상 속도와 원래 피치로 전체 곡을 재생하여 진행 상황을 확인합니다.

### 고급 워크플로우: 합주 준비

사용자는 밴드 음원을 Music Trainer에 업로드합니다. 앱이 음원을 보컬, 드럼, 베이스, 기타, 키보드로 분리합니다. 자신은 베이스 연주자이므로 보컬과 키보드만 음소거하고 드럼, 기타와 함께 베이스 라인을 연습합니다. 어려운 부분은 A-B 루프로 표시하여 느린 속도(0.7x)로 반복 연습합니다. 준비가 되면 모든 악기를 활성화하여 전체 앙상블에서의 자신의 부분을 연습합니다.

### 교육자 워크플로우: 학생 맞춤 학습

음악 교육자는 학생의 수준에 맞추어 곡의 속도를 조절합니다. 초급자는 0.7x로 늦춘 버전을 사용하고, 고급자는 1.2x로 빠른 버전을 사용합니다. 피치 제어로 학생의 음역대에 맞춥니다. 합주 곡의 경우, 학생이 담당하지 않은 악기는 음소거하여 학생의 파트 집중력을 높입니다.

## 핵심 차별화 요소

### 속도/피치/볼륨이 완전히 독립적

일반 오디오 플레이어와 달리 Music Trainer는 세 가지 파라미터가 완전히 독립적입니다:
- 속도 변경 시 피치는 자동 유지
- 피치 변경 시 재생 속도는 자동 유지
- 각 악기의 볼륨을 개별 제어

### 전문 음악 소프트웨어 수준의 인터페이스

Logic Pro, Ableton Live 등 전문 음악 소프트웨어의 다크 테마와 컨트롤 레이아웃을 모방하여:
- 직관적이고 전문적인 외관
- 음악 전공자에게 친숙한 워크플로우
- 고급 기능도 쉽게 접근 가능

### AI 기반 음원 분리

Demucs를 활용한 고급 음원 분리 기술:
- 일반 스마트폰 오디오 플레이어로 불가능한 기능
- 합주 곡에서 특정 악기만 학습 가능
- 음악 교육의 새로운 가능성

### 웹 기반 플랫폼

다운로드 없이 즉시 사용:
- 모든 브라우저에서 동일한 경험
- 여러 디바이스에서 자유롭게 접근
- 음악 학습 시간을 최소한의 장벽으로

## 성공 지표

**사용자 관점**
- 일주일 활성 사용자 수(WAU) 1,000명 이상
- 평균 세션 길이 30분 이상
- 월간 반복 사용율 60% 이상
- 사용자 만족도 4.5/5.0 이상

**기술 관점**
- 페이지 로드 시간 2초 이내
- 재생 끊김 없음(0.00% 버퍼링 에러)
- 무끊김 속도/피치 변경
- 모바일/데스크톱 크로스 브라우저 호환성 100%

**비즈니스 관점**
- Phase 1: MVP 1개월 내 배포
- Phase 2: 프리미엄 전환율 5% 이상
- Phase 3: 구독 ARPU(평균 구독 수익) $2.99/월 이상
